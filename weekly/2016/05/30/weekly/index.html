Node.js 주간 뉴스 2016년 5월 30일

  • 작성자: Minwoo Jung (github.com/JungMinu)
  • 원문: Weekly Update - May 30th, 2016
  • 번역: yous

Node v4.4.5(LTS) 릴리스

Node v4.4.5(LTS)가 릴리스되었습니다. 전체 변경사항은 GitHub에서 보실 수 있습니다.

커뮤니티 업데이트

  • NodeUp 팟캐스트 에피소드 102: Anna Henningsen, James Snell, Rich Trott, Rod Vagg가 함께 진행하는 Node v6.0 쇼.
  • Node Hero - Node.js 시작하기: Node.js 입문과 더불어 Node.js를 이용해 어떻게 소프트웨어 제품을 배포할 수 있는지 배울 수 있습니다.
  • Node.js 코어 소스를 읽는 빠른 가이드: Node.js 코어를 구성하는 소스 코드를 이해하는 접근 방식 중 하나입니다. “사람마다 차이가 있을 수 있습니다. 개봉하면 보증은 무효가 됩니다.”

Node.js에 관련한 글을 쓰거나 발견했다면, Evangelism 팀 저장소의 이슈 페이지에 주간 업데이트 이슈로 알려주세요.

다가오는 이벤트

  • NodeConf 브라질 - 2016년 6월 4일부터 5일까지 열립니다.
  • NodeConf Adventure 2016, “NodeConf Adventure 첫 예매가 시작됩니다!”, 미국 캘리포니아 주 마린 카운티 워커 크릭 랜치에서 2016년 6월 9일부터 12일까지 열립니다.
  • Node PDX - 2016년 6월 19일부터 21일까지 열립니다.
  • NodeConf 아르헨티나 - 2016년 11월 18일부터 19일까지 열립니다.

계획 중인 Node.js 이벤트가 있나요? Evangelism 팀 저장소의 이슈 페이지에 주간 업데이트 이슈를 통해 알릴 수 있습니다.
